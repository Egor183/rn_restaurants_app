## Description

This project was created to develop skills in React Native, TS, state management, increasing query performance, adding base mobile features, and testing.

## Main technologies

React Native, Redux Toolkit, TS, React Query, Jest

## How to launch

### IOS

1. yarn
2. cd ios && pod install
3. start build project using Xcode

### Android

1. yarn
2. https://stackoverflow.com/questions/27620262/sdk-location-not-found-define-location-with-sdk-dir-in-the-local-properties-fil
3. yarn android
